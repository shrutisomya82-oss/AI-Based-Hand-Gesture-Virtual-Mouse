# AI Hand Gesture Virtual Mouse 🖱️🤖

An AI-powered virtual mouse that allows you to control your computer cursor, click, drag, scroll, and take screenshots using real-time hand gestures. Built with Python, OpenCV, MediaPipe, and PyAutoGUI.

## Features
- **Smooth Cursor Control:** Real-time pointer control with a custom smoothing filter to prevent jitter.
- **Dynamic HUD:** On-screen visual feedback and gesture status.
- **Control Panel GUI:** Multi-threaded Tkinter desktop interface to adjust sensitivity and start/stop tracking in real-time.
